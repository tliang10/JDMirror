# 职镜 JDMirror

> 看清JD，照见自己 —— 用 AI 翻译招聘 JD，分析简历匹配度

职镜 JDMirror 是一款 Chrome 浏览器插件，帮助求职者快速理解招聘 JD 的真实含义，并评估自己与岗位的匹配程度。

---

## 安装指南

### 第一步：获取插件压缩包

插件已打包为 `jd_mirror.zip` 压缩包，将其解压到本地任意目录（请勿删除解压后的文件夹，否则插件将失效）。

### 第二步：打开 Chrome 扩展管理页面

在 Chrome 浏览器地址栏输入以下地址，或通过菜单进入：

```
chrome://extensions/
```

> 菜单路径：右上角 `⋮` → 更多工具 → 扩展程序

### 第三步：开启开发者模式

在扩展管理页面右上角，打开 **「开发者模式」** 开关。

![开发者模式](https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=A%20screenshot-style%20image%20showing%20the%20Chrome%20extensions%20management%20page%20with%20the%20Developer%20Mode%20toggle%20switch%20turned%20on%20in%20the%20top%20right%20corner,%20clean%20UI,%20light%20theme&image_size=landscape_16_9)

### 第四步：加载插件

1. 点击 **「加载已解压的扩展程序」** 按钮
2. 在弹出的文件选择窗口中，选择刚才解压的 `jd_mirror` 文件夹
3. 点击「选择文件夹」确认

### 第五步：确认安装成功

安装成功后，Chrome 工具栏右上角会出现职镜 JDMirror 的图标（蓝色镜子图标）。同时可以在扩展程序列表中看到「职镜JDMirror」。

> **提示**：建议将插件图标固定到工具栏，方便随时使用。点击右上角拼图图标 🧩，找到职镜 JDMirror，点击图钉 📌 固定。

## 使用指南

### 核心功能一览

| 功能 | 说明 | 适用场景 |

---

### 功能一：照JD（JD 解读）

**做什么**：将招聘网站上晦涩的 JD "翻译"成人话，帮你一眼看懂这个岗位到底是干什么的。

**操作步骤**：

1. 打开任意招聘网站的岗位详情页（支持 BOSS直聘、猎聘、智联招聘、拉勾、51job 等）
2. 点击浏览器工具栏中的 **职镜 JDMirror 图标**
3. 在弹出的面板中，点击 **「照JD」** 按钮
4. 插件会自动提取页面中的 JD 文本，并调用 AI 进行分析
5. 等待几秒，AI 会流式输出分析结果

**分析结果包含 6 个模块**：

| 模块 | 内容 |
|------|------|
| 📋 岗位职责 | 这个岗位日常真正在做什么（大白话版） |
| ✅ 显性要求 | JD 中明确写出的学历、经验、技能要求 |
| 🔍 隐性要求 | JD 没明说但实际存在的潜规则（如"能承受压力"=加班多） |
| ⏰ 常见日常工作 | 典型的一天工作安排，按时间线展示 |
| 📊 典型 KPI | 入职后老板用什么指标考核你 |
| 💬 一句话总结 | 用最通俗的话概括这个岗位 |

**交互提示**：
- 每个模块以折叠面板展示，点击标题可展开/收起
- KPI 模块以卡片+进度条形式可视化展示
- 日常工作以时间线形式呈现
- 如果自动提取的 JD 不准确，可以在文本框中手动编辑后点击「重新分析」

---

### 功能二：照自己（匹配分析）

**做什么**：上传你的简历，AI 会对比 JD 要求，给出匹配度评分、能力雷达图和面试突击指南。

**操作步骤**：

#### 首次使用（上传简历）

1. 点击浏览器工具栏中的 **职镜 JDMirror 图标**
2. 在弹出的面板中，点击 **「照自己」** 按钮
3. 进入简历上传页面，支持两种上传方式：
   - **拖拽上传**：将简历文件直接拖入上传区域
   - **点击上传**：点击「选择文件」按钮，从电脑中选择简历
4. 支持的格式：**PDF、DOCX、DOC、TXT**（最大 5MB）
5. 上传成功后，如果当前页面是招聘岗位页，会自动跳转到匹配分析

#### 再次使用（已上传简历）

1. 打开任意招聘网站的岗位详情页
2. 点击插件图标，点击 **「照自己」**
3. AI 会自动提取当前页面 JD，并结合你的简历进行匹配分析

**分析结果包含**：

| 模块 | 内容 |
|------|------|
| 🎯 综合匹配度 | 0-100% 环形图，红/黄/绿三色评级 |
| 🕸️ 能力雷达图 | 5-6 个动态维度的可视化对比（带动画） |
| 📈 维度分析 | 每个维度的评分进度条 + 详细评语 |
| ✅ 优势分析 | 你的简历在哪些方面突出 |
| ⚠️ 不足分析 | 与 JD 要求的差距在哪里 |
| 📝 面试突击指南 | 按优先级排列的短期准备建议（含学习内容和时间估算） |

**简历管理**：
- 简历上传后会持久化保存，无需每次重新上传
- 点击「更换简历」可重新上传新的简历文件
- 底部状态栏会显示当前简历状态

---

### 功能三：保存分析报告

分析完成后，你可以将结果导出保存，方便离线查看或分享。

**操作步骤**：

1. 在 JD 解读页面或匹配分析页面，点击 **「保存」** 按钮
2. 在弹出的对话框中，选择导出格式：
   - **HTML**：带样式的网页报告，浏览器直接打开
   - **Markdown**：纯文本格式，可导入 Notion、Obsidian 等笔记工具
3. 文件会自动下载，命名格式为：`岗位名称_JD解读_日期_时间戳.html`

**报告内容**：
- 原始 JD 文本
- JD 解读（6 个模块）
- 匹配分析（综合匹配度、维度分析、优势/不足、突击指南）
- 原始页面链接

### 常见问题

**Q: 点击「照JD」后提示"未检测到JD内容"？**
A: 请确保当前页面是招聘岗位的**详情页**（不是列表页）。如果确认是详情页但仍无法识别，可以手动复制 JD 文本粘贴到文本框中。

**Q: 简历上传后提示"解析失败"？**
A: 请检查：
- 文件大小是否超过 5MB
- PDF 是否为扫描件（纯图片 PDF 无法提取文字，请使用可编辑的 PDF）
- DOC 旧格式建议用 Word 另存为 DOCX 后重新上传

**Q: 分析结果可以保存吗？**
A: 可以。点击分析结果页面的「保存」按钮，支持导出为 HTML 或 Markdown 格式。

**Q: 需要自己配置 API Key 吗？**
A: 插件已预置默认 API Key，开箱即用。如需使用自己的 Key 或切换模型，可在设置中修改。
